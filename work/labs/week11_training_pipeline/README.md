# Week 11: Self-Refine 訓練數據生成流程

## 🎯 學習目標

1.  **理解 Self-Refine 作為數據增強工具**：學習如何利用「生成-評估-改進」的循環來創造高品質、多樣化的訓練數據。
2.  **建立自動化數據收集管道**：實作一個可以自動執行多輪迭代，並將結果儲存為結構化數據（JSONL）的流程。
3.  **區分流程目標**：深刻理解「產出最終內容」與「產出訓練數據」這兩種不同目標在架構設計上的差異。

## 🚀 專案架構

```
week11_training_pipeline/
├── main.py                     # 主執行腳本，包含 SelfRefineCrew 的核心邏輯
├── agent_definitions.py        # 集中定義 ContentCreator 和 Critique Agent
├── training_data_collector.py  # 負責收集和儲存訓練數據的類別
└── training_data.jsonl         # (輸出) 生成的訓練數據檔案
```

## 核心流程

1.  **初始生成**：`ContentCreatorAgent` 根據給定主題生成一份初始內容。
2.  **迭代循環 (N次)**：
    a.  **評估**：`CritiqueAgent` 對當前內容進行評估，給出分數和改進建議。
    b.  **改進**：`ContentCreatorAgent` 根據評估建議，生成一個改進後的新版本。
    c.  **數據收集**：`TrainingDataCollector` 捕獲這一輪的 `(改進指令, 改進後內容)` 作為一個數據點。
3.  **儲存**：所有收集到的數據點被儲存到 `training_data.jsonl` 檔案中。

## 🆚 與 Week 02 (進階反思) 的主要差異

雖然本週的專案和 `week02/advanced_solution.py` 都使用了自我反思（Self-Refine）的模式，但它們的**核心目標**和**架構設計**有著根本性的不同。

| 特性 | Week 02 (進階反思) | Week 11 (訓練數據生成) |
| :--- | :--- | :--- |
| **核心目標** | **產出單一、高品質的最終內容**。 | **產出大量、結構化的訓練數據**。 |
| **最終產出** | 一份經過多輪優化的**最終文案 (字串)**。 | 一個 `training_data.jsonl` **檔案**。 |
| **關鍵組件** | `AdvancedReflectionWorkflow`：一個複雜的類別，在內部管理所有迭代和評估邏輯。 | `TrainingDataCollector`：一個簡單、專注的**數據收集器**，在主流程外部被動地記錄數據。 |
| **流程重點** | 過程的**精細控制**和**動態配置**（例如，不同的評估標準和品質閾值）。 | 過程的**自動化**和最終數據的**格式化**（生成 `prompt`/`completion` 對）。 |
| **比喻** | 一位**工匠**，精心雕琢一件藝術品。 | 一條**生產線**，高效地生產標準化的零件（訓練數據）。 |

總之，`Week 02` 展示瞭如何使用反思模式來**解決單一問題**，而 `Week 11` 展示瞭如何利用相同的模式來**為解決未來更多問題（模型微調）做準備**。這是一個從「應用」到「基礎設施」的思維轉變。

## ⚙️ 如何執行

### 1. 生成訓練數據

首先，執行 `main.py` 來啟動 Self-Refine 流程並生成 `training_data.jsonl` 檔案。

```bash
python main.py
```

### 2. 檢視數據並展示基準性能

接著，執行 `train.py`。這個腳本現在的用途是：
1.  讀取並驗證 `training_data.jsonl`。
2.  打印一條訊息，說明這些數據已準備好用於**外部的標準微調流程**。
3.  執行一個未經訓練的「基準 Crew」，展示其性能，作為未來微調改善的對照組。

```bash
python train.py
```

**⚠️ 重要說明**: 經過進一步測試，我們確認 `crewai` 的 `train()` 方法強烈依賴一個包含 `human_feedback` 欄位的特定內部數據格式，該格式無法由我們的自動化流程生成。因此，**我們不再調用 `crew.train()`**。本 Epic 的重點是**生成可用於標準微調工具的高品質數據集**，而不是使用 `crewai` 內部的實驗性訓練功能。
